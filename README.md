# M-Audio Axiom 61 DirectLink for Bitwig Studio

Controller script for full two-way integration of M-Audio Axiom 61 with Bitwig Studio via DirectLink protocol.

**Documentation:** [English](README_EN.md) | [Русский](README_RU.md)

## Quick Start

1. Copy `Axiom61DirectLink.control.js` to your Bitwig Controller Scripts folder
2. Add controller in Bitwig: **Settings** → **Controllers** → **M-Audio** → **Axiom 61 DirectLink**
3. Configure MIDI ports (Input 1: Axiom 61, Input 2: DirectLink Out)
