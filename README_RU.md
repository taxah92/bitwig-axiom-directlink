# M-Audio Axiom 61 DirectLink для Bitwig Studio

Контроллер-скрипт для полноценной двухсторонней интеграции M-Audio Axiom 61 с Bitwig Studio по протоколу DirectLink.

## Возможности

### Микшер
- **Слайдеры 1-8**: Управление громкостью 8 треков
- **Master слайдер**: Управление мастер-громкостью
- **Кнопки Mute/Solo/Record**: Переключение Mute/Solo/Record для треков
- **Энкодеры**: Управление панорамой, параметрами устройств или посылами

### Транспорт
- **Play**: Воспроизведение/пауза
- **Stop**: Остановка
- **Record**: Запись
- **Loop**: Переключение Loop режима
- **Rewind/Fast Forward**: Перемотка (удержание)

### Навигация
- **Prev/Next Track**: Переход к предыдущему/следующему треку
- **Bank Up/Down**: Прокрутка банка треков

### Дисплей
- Отображение имени выбранного трека
- Отображение текущего режима энкодеров

### Двухсторонняя синхронизация
- Индикаторы слайдеров отображают текущую громкость
- LED кнопок отображают состояния Mute/Solo/Record
- Transport LEDs синхронизированы с DAW

## Установка

### Шаг 1: Копирование скрипта

Скопируйте файл `Axiom61DirectLink.control.js` в директорию контроллеров Bitwig Studio:

**Windows:**
```
%USERPROFILE%\Documents\Bitwig Studio\Controller Scripts\
```

**macOS:**
```
~/Documents/Bitwig Studio/Controller Scripts/
```

**Linux:**
```
~/Bitwig Studio/Controller Scripts/
```

### Шаг 2: Настройка MIDI портов

1. Подключите M-Audio Axiom 61 к компьютеру
2. Запустите Bitwig Studio
3. Откройте **Settings** → **Controllers**
4. Нажмите **Add Controller**
5. Выберите **M-Audio** → **Axiom 61 DirectLink**
6. Убедитесь, что MIDI порты настроены правильно:
   - **Input 1**: Axiom 61 (стандартный MIDI - клавиши, пэды, контроллеры)
   - **Input 2**: DirectLink Out (ответы от контроллера)
   - **Output 1**: Axiom 61 (стандартный MIDI)
   - **Output 2**: DirectLink Out (индикаторы и дисплей)

### Шаг 3: Активация

1. Включите скрипт (кнопка "On/Off")
2. На контроллере должен отобразиться текст "BITWIG" на дисплее
3. Индикаторы должны синхронизироваться с DAW

## Управление

### Пэды (DirectLink режим)

| Пэд | Без Shift | С Shift |
|-----|-----------|---------|
| 1 | Запуск клипа | PAN режим (энкодеры) |
| 2 | Запуск клипа | DEVICE режим (энкодеры) |
| 3 | Запуск клипа | SEND режим (энкодеры) |
| 4 | Запуск клипа | Toggle Metronome |
| 5 | Запуск клипа | Undo |
| 6 | Запуск клипа | Redo |
| 7 | Запуск клипа | Tap Tempo |
| 8 | Запуск клипа | Toggle Loop |

### Кнопки Mute/Solo/Record

- **Обычный режим**: Mute/Solo/Record в зависимости от текущего режима
- **Shift + кнопка 1-8**: Выбор трека
- **Кнопка 9 или Flip**: Переключение режима Mute → Solo → Record → Mute

### Настройки (Preferences)

В Bitwig Studio: **Settings** → **Controllers** → **Axiom 61 DirectLink**

Доступные настройки:
- **Button Mode**: Выбор режима кнопок (Mute/Solo/Record)

### Навигация

- **Prev/Next Track**: Выбор предыдущего/следующего трека
- **Bank Up/Down**: Прокрутка банка из 8 треков
- **Shift + Bank Up/Down**: Навигация по сценам

## Технические детали

### MIDI порты

| Порт | Название | Направление | Назначение |
|------|----------|-------------|------------|
| 0 | Axiom USB Out | Input/Output | Стандартный MIDI (клавиши, пэды, контроллеры) |
| 1 | DirectLink Out | Input/Output | DirectLink протокол (индикаторы, дисплей, ответы) |

### SysEx команды

```
Включение DirectLink:    F0 00 01 05 20 7F 20 2E F7
Отключение DirectLink:   F0 00 01 05 20 7F 20 00 F7
Очистка дисплея:         F0 00 01 05 20 7F 10 F7
Вывод текста:            F0 00 01 05 20 7F 11 LL PP PP <data> F7
```

### CC маппинг

| CC | Назначение |
|----|------------|
| 17-24 | Энкодеры (relative mode) |
| 33-40 | Слайдеры 1-8 |
| 41 | Master слайдер |
| 49-56 | Кнопки Mute/Solo |
| 110-111 | Навигация по трекам |
| 113-118 | Transport (Loop, RW, FF, Stop, Play, Rec) |

## Устранение неполадок

### Скрипт не определяется

1. Убедитесь, что файл находится в правильной директории
2. Перезапустите Bitwig Studio
3. Проверьте, что имя файла имеет расширение `.control.js`

### Индикаторы не работают

1. Убедитесь, что DirectLink Out порт правильно настроен
2. Проверьте, что контроллер находится в режиме DirectLink
3. На Axiom 61 должен быть включен режим DirectLink (см. мануал контроллера)

### Дисплей не показывает текст

1. Проверьте подключение DirectLink Out порта
2. Убедитесь, что скрипт успешно инициализировался (проверьте консоль)

### Энкодеры работают некорректно

1. Убедитесь, что энкодеры настроены в Relative mode
2. На Axiom 61: Settings → Encoder Mode → Relative
